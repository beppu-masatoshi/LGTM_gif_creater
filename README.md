# LGTM_gif_creater

GIF画像に「LGTM」などの文字を自動で焼き込むツールです。
アニメーションGIFにも対応しており、文字の**位置**と**表示するタイミング(何秒後から何秒後まで表示するか)**を自由に指定できます。

- `lgtm.py` : コマンドラインから使うツール
- `gui.py` : ドラッグ&ドロップで使えるGUIアプリ(プレビューでGIFが実際に再生され、クリックやスライダーで位置・タイミングを確認しながら調整できます)

## 事前準備

Python 3.10以上が必要です。以下のライブラリをインストールしてください。

```bash
pip install Pillow tkinterdnd2
```

- `Pillow` : 画像処理に必須です。
- `tkinterdnd2` : GUIでのドラッグ&ドロップに使います(任意。無くてもGUIは起動しますが、ドラッグ&ドロップは無効になり、ファイル選択ボタンのみ使えます)。

## 使い方 (CLI: `lgtm.py`)

```bash
python lgtm.py 入力.gif 出力.gif
```

### オプション

| オプション | 説明 | デフォルト |
| --- | --- | --- |
| `--text` | 焼き込む文字列 | `LGTM` |
| `--color` | 文字の塗り色 | `white` |
| `--outline` | 文字の縁取り色 | `black` |
| `--x` | 文字の水平位置 (0=左端 〜 100=右端) | `50` (中央) |
| `--y` | 文字の垂直位置 (0=上端 〜 100=下端) | `50` (中央) |
| `--start` | 文字を表示し始める時間 (ミリ秒) | `0` (最初から) |
| `--end` | 文字の表示を終える時間 (ミリ秒) | 指定なし (最後まで表示) |

### 例

```bash
# 標準的なLGTMスタンプ
python lgtm.py input.gif output.gif

# 文字・色・縁取り色を変更
python lgtm.py input.gif output.gif --text "NICE" --color yellow --outline black

# 右下に表示
python lgtm.py input.gif output.gif --x 90 --y 90

# 0.5秒後〜2秒後の間だけ文字を表示
python lgtm.py input.gif output.gif --start 500 --end 2000
```

## 使い方 (GUI: `gui.py`)

```bash
python gui.py
```

1. ウィンドウにGIFファイルをドラッグ&ドロップするか、「ファイルを選択...」ボタンから選びます。
2. プレビュー欄に選択したGIFがそのまま再生されます。プレビューを**クリック/ドラッグ**すると、その場所に文字の位置を自由に移動できます(「中央に戻す」ボタンでリセット可能)。
3. 「文字」「文字色」「縁色」を好きな内容に変更できます(即座にプレビューへ反映されます)。
4. 「表示タイミング」の**開始・終了スライダー**で、アニメーション内で文字を表示する時間帯を指定できます。スライダーをドラッグしている間はその時点の映像で確認でき、離すと自動再生に戻ります。
5. 「変換する」ボタンを押すと、選択したGIFと同じフォルダに `元のファイル名_LGTM.gif` として保存されます。

## 補足

- 変換元のGIFのフレーム表示時間・ループ回数・透過情報はそのまま保持されます。
